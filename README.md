# Books Scraper Project

## Цель проекта
Сбор информации о книгах с сайта [Books to Scrape](http://books.toscrape.com) для анализа, хранения и последующей обработки.  
Проект автоматически собирает данные о каждой книге: название, цену, рейтинг, наличие, описание и дополнительные характеристики.

## Инструкции по запуску

1. Клонируйте репозиторий или скопируйте проект на локальный компьютер.
2. Создайте виртуальное окружение и активируйте его (Windows):

```bash
python -m venv venv
.\venv\Scripts\activate
```
    
4. Установите зависимости из файла requirements.txt:

 ```bash
    pip install -r requirements.txt
   ```
5. Запуск скрипта для парсинга книг:

```bash
python scraper.py
```

6. Запуск автотестов:
```bash
pytest
```
